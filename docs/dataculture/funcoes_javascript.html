Funções em JavaScript

Explore os exemplos abaixo — edite os campos e veja o resultado em tempo real.

Definição function declaration
function saudar(nome) { return `Olá, ${nome}!`; }
Teste ao vivo
saudar("Maria");
→ "Olá, Maria!"

Em JS, usamos function no lugar de def, chaves {} delimitam o bloco, e template literals `${variavel}` facilitam montar strings.

Definição múltiplos parâmetros
function calcularArea(largura, altura) { const area = largura * altura; return area; }
Teste ao vivo
calcularArea(5, 3);
→ 15

Usamos const para variáveis locais que não mudam. JS também tem let (pode mudar) e var (mais antigo, evite). Os ponto-e-vírgula são opcionais, mas recomendados.

Definição decisão com if/else
function verificarAcesso(idade) { if (idade >= 18) { return "Acesso liberado!"; } else { return "Acesso negado."; } }
Teste ao vivo
verificarAcesso(20);
→ "Acesso liberado!"

A lógica é idêntica ao Python — a diferença está na sintaxe: condições ficam entre parênteses () e os blocos entre chaves {}.

Forma clássica vs. arrow function
// forma clássica function dobrar(n) { return n * 2; } // arrow function — mesma coisa, mais curta const dobrar = (n) => n * 2;
Teste ao vivo
dobrar(7);
→ 14

Arrow functions são um atalho do JS moderno. Quando a função tem só uma linha, você pode omitir o return e as chaves — o resultado é retornado automaticamente. Você vai ver muito esse estilo no dia a dia.
